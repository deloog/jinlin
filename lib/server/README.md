# 服务端代码

这个文件夹包含所有服务端相关的代码，可以直接复制到服务器上使用。

## 文件夹结构

- `database/` - 数据库模型和操作
- `api/` - API接口定义和实现
- `models/` - 数据模型
- `services/` - 服务层
- `utils/` - 工具函数

## 部署说明

1. 安装Node.js和npm
2. 复制整个`server`文件夹到服务器
3. 运行`npm install`安装依赖
4. 配置数据库连接（见`config.js`）
5. 运行`npm start`启动服务

## 开发模式

在开发过程中，可以使用以下命令启动服务：

```
node server.js
```

服务器默认运行在3000端口。
